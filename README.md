## ✅ 正誤表（Errata）
- p. 132: Rコードの実行において

>fit <- fitdistr(x, "gamma")

の前に

>x <- kikou$風速

を実行する必要があります。このとき、出力結果は
```r
> confint(fit) # 95%信頼区間
          2.5 %    97.5 %
shape 10.147950 13.536898
rate   3.721611  4.995714

> confint(fit, level = 0.99) # 99%信頼区間
         0.5 %   99.5 %
shape 9.615507 14.06934
rate  3.521435  5.19589
```
のようになるはずです。
